# 卡牌摆放互动页（纯 HTML/CSS/JS）

## 功能概览
- 12 张性格卡牌、24 个牌面（正反面），内置数据，直接打开 `index.html` 即可使用。
- 左侧卡组侧边栏：
  - 纵向滚动列表，展示牌面图片（`public/<中文标题>.png`）。
  - 点击切换正/反面，拖拽未使用的牌到右侧空位。
  - 已放置的卡牌不再显示，放完显示“已全部放置”。
- 右侧摆放区：
  - 4×3 网格，空位提示“空位”。
  - 放置后显示选定面的大图；点击可在正/反面间切换，拖拽可交换位置或移到空位。
- 控制按钮：
  - **算牌**：按行加权计算分数（第1行+2，第2行+1，第3行+0），按颜色汇总，并给出等级（>17 超级，10–17 明显，其余一般）。
  - **重置**：清空布局并恢复卡组。
  - **导出当前布局**：在控制台打印 12 长度数组（包含 cardId/side/title/english/value/color/slotIndex）。

## 运行方式
1. 确保 `public` 目录下存在与牌面中文标题一致的 `.png` 文件（已 URL 编码处理）。
2. 双击或用浏览器打开 `index.html`，无任何构建依赖。

## 目录结构
- `index.html`：主页面，内联 CSS/JS。
- `public/`：牌面图片，文件名需与牌面中文标题一致，后缀 `.png`。
- `pro.md`：需求文档。

## 主要交互说明
- 左侧卡组：点击切换正/反，拖拽到右侧空位完成放置。
- 摆放区：点击卡牌翻面；拖拽卡牌可交换或移到空位。
- 算牌：点击按钮查看各色分数与等级；重置可重新开始。
